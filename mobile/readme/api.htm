ShopNC【B2B2C】平台系统接口文档

数据格式约定

返回数据为json格式，code为200表示数据正常，code为400表示数据出错，datas为返回的数据 

{code: 200, datas: {}}

分页参数

支持分页的接口，拼接以下参数到请求URL
  • page 每页商品数量(默认5)
  • curpage 当前页数

返回值

获取数据

//有内容直接返回数组，无内容返回空数组[]
//hasmore为是否有下一页
//page_total为总页数
{"code":200,"hasmore":true,"page_total":10,"datas":{"goods_list":[]}}

操作成功

//code返回code为200，datas返回"1"
{"code":200,"datas":{"1"}}

错误

//出现错误返回code为400，error内为错误信息
{"code":400,"datas":{"error":"参数错误"}}
//登录失败，login:0
{"code":400,"login":"0","datas":{"error":"\u8bf7\u767b\u9646"}}

商品列表

商品列表接口(get)

  index.php?act=goods&op=goods_list

请求参数

  • key 排序方式 1-销量 2-浏览量 3-价格 空-按最新发布排序
  • order 排序方式 1-升序 2-降序
  • page 每页数量
  • curpage 当前页码
  • gc_id 分类编号
  • keyword 搜索关键字
  • barcode 商品条形码
  • b_id 品牌id
  • gc_id、keyword、barcode、b_id 四选一不能同时出现

返回数据

  • goods_id 商品编号
  • goods_name 商品名称
  • goods_price 商品价格
  • goods_marketprice 商品市场价
  • goods_salenum 销量
  • evaluation_good_star 评价星级
  • evaluation_count 评价数
  • sole_flag 是否手机专享
  • group_flag 是否团购
  • xianshi_flag 是否限时折扣
  • goods_image 图片名称
  • goods_image_url 图片地址
  • is_fcode 是否为F码商品 1-是 0-否
  • is_appoint 是否是预约商品 1-是 0-否
  • is_presell 是否是预售商品 1-是 0-否
  • have_gift 是否拥有赠品 1-是 0-否
